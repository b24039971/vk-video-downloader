# VK & TikTok Video Downloader

Простой и удобный скрипт на Python для скачивания видео из ВКонтакте (VK) и TikTok в высоком качестве. Под капотом используется мощная библиотека `yt-dlp`.

## ✨ Возможности

- Скачивание видео из **VK** (vk.com, vkontakte.ru) и **TikTok** (tiktok.com, vm.tiktok.com, vt.tiktok.com).
- Автоматический выбор наилучшего доступного качества (видео ≥ 720p + лучшее аудио).
- Интерактивный режим работы (ввод ссылок по очереди) и пакетный (через аргументы командной строки).
- Информативный прогресс-бар загрузки (скорость, размер, проценты).
- Автоматическое создание папки `downloads/` для сохранения видео.
- Безопасная обработка названий файлов для совместимости с Windows.

## 🛠 Требования

- Python 3.7+
- Установленный пакет [yt-dlp](https://github.com/yt-dlp/yt-dlp) (включён в `requirements.txt`).
- **FFmpeg**: Для объединения видео и аудио потоков высокого качества `yt-dlp` требует FFmpeg.
  - **Windows**: Скачайте [FFmpeg](https://ffmpeg.org/download.html), распакуйте и добавьте путь к папке `bin` в переменные среды (PATH).
  - **macOS**: `brew install ffmpeg`
  - **Linux (Ubuntu/Debian)**: `sudo apt install ffmpeg`

## 🚀 Установка

1. Склонируйте репозиторий:
   ```bash
   git clone https://github.com/your-username/vk-video-downloader.git
   cd vk-video-downloader
   ```

2. (Опционально) Создайте и активируйте виртуальное окружение:
   ```bash
   python -m venv venv
   # На Windows:
   venv\Scripts\activate
   # На macOS/Linux:
   source venv/bin/activate
   ```

3. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```

## 💻 Использование

### Интерактивный режим (рекомендуется)
Просто запустите скрипт без аргументов. Программа будет запрашивать ссылки одну за другой:
```bash
python vk_video_downloader.py
```

### Скачивание одного видео из консоли
Передайте ссылку как аргумент при запуске:
```bash
python vk_video_downloader.py "https://vk.com/video-12345678_90123456"
```
